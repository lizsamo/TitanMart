## App Overview

**TitanMart** is a consumer-to-consumer marketplace designed exclusively for **California State University, Fullerton** students.  
It enables students to safely **buy and sell items** within a trusted campus environment. Providing an authenticated, student-only platform for secondhand goods, textbooks, electronics, dorm essentials, and more.

---

## Project Goals

The goal of this project is to provide a **secure, cloud-powered marketplace** that promotes trust and convenience among CSUF students.  
TitanMart ensures safety through structured payments, verified campus authentication, and transparent user feedback systems.

---

## Key Features

- **Student Authentication** — Only CSUF students can sign up using university credentials.  
- **Product Listings** — Post, browse, and search for campus-specific items.  
- **Secure Checkout** — Structured payment processing through integrated APIs.  
- **Ratings & Reviews** — Promote trust and accountability between buyers and sellers.  
- **Cloud-Hosted Backend** — Database, file storage, and API endpoints hosted on a cloud platform.  
- **Cloud-Native Security Controls** — IAM roles, WAF policies, and misconfiguration alerts for secure operation.

## Deployment Overview

1. **Frontend (iOS App)**  
   - Built and deployed via **Xcode** using **SwiftUI**.  
   - Communicates with backend REST APIs over HTTPS.  

2. **Backend (Cloud API)**  
   - Deployed to AWS (or GCP/Azure) using managed services.  
   - Includes API routes for user authentication, product management, and payment handling.

3. **Database & Storage**  
   - Centralized cloud-hosted database.  
   - Secure image uploads and product listing storage.

4. **Security Controls**  
   - IAM roles restrict access by service.  
   - WAF rules protect endpoints from common exploits.  
   - Continuous monitoring detects misconfigurations or unusual access behavior.
